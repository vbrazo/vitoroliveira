# vitoroliveira.ca
Design System
 - Public entry: `src/design-system/index.ts`
 - Re-exports shadcn/ui primitives and adds DS atoms like `Container`, `Section`, and `Typography`.
 - Use imports from `@/design-system` rather than deep `@/components/ui/*` in feature code when possible.
## What technologies are used for this project?

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
